HTML & CSS Quick Reference Guide





HTML Tags

Html

This tag wraps all the content of the HTML dcoument

 <!DOCTYPE html>
        <html>
        <head>
        <link rel="stylesheet" href="guide.css" />
        <meta charset="UTF-8" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <title>Page Title</title>
        </head>
        <body>
        
        <h1> my First Heading</h1>
        <p> my first paragraph.</p>
        
        <body>
        </html>

Head

This tag contains the styles, links, and metadata to external resources

      <html>

      </head>

      </html>

a

This tag creates hyperlinks to other webpages/sites as well as sections of the current page

 <!DOCTYPE html>
      <html>
      <head>
      <link rel="stylesheet" href="guide.css" />
      <meta charset="UTF-8" />

      <body>
      </html>

ul, ol, & li

These tags are used to create unordered and ordered lists


      <body>
      <ul>
      <li>Coffee</li>
      <li>Tea</li>
      <li>Milk</li>
      </ul>

      <ol>
        <li>Coffee</li>
        <li>Tea</li>
        <li>Milk</li>
        </ol>
      <body>

CSS Properties

Color

This sets the color of an element


      .text {
        Color: black;
        background-color: white;
        font-size: 30px;
        font-family:'Courier New', Courier, monospace;
        margin: 10px;
        padding: 2px;
        width: 10px;
        height: 5px;
        display: block;
        position: absolute;
        text-align: center;
      }
